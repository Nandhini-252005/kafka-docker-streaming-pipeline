# Kafka Docker Streaming Pipeline

This project provides a simple and reliable Docker-based setup for running **Apache Kafka** and **Zookeeper** locally using **Docker Compose**. It is designed for learning, development, and prototyping real-time data streaming applications.

---

## Features

- Dockerized Kafka and Zookeeper setup
- Easy one-command startup using Docker Compose
- Suitable for real-time streaming demos and learning Kafka basics
- Compatible with Kafka producers and consumers written in any language

---

## Technologies Used

- Apache Kafka
- Apache Zookeeper
- Docker
- Docker Compose
- Confluent Kafka Docker Images
